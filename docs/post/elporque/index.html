Linux ¿Por qué?

¿Por qué migré a linux?

   

Después de notar que las versiones de windows a partir de la versión de windows “vista” incrementaba notablemente el consumo en recursos, y cada vez, se hacía necesario el uso programas de protección de virus, spyware y malware, entre otros. Hacía que el uso de mi sistema para tareas cotidianas se conviertan en un dolor verdadero de cabeza.

Junto a todo esto,el paso del tiempo hacia que el sistema se sintiera mucho mas lento, obligando a su vez a realizar el respaldo, formateo y reinstalación del sistema, para así mantener un sistema de windows un poco más rápido.

Las actualizaciones cada vez eran más intrusivas, casi imponiéndose a la decisión del usuario. Esto me provocaba con usuario sentir que perdía cada vez más el control sobre el propio sistema. Estos eran los principales factores que afrontaba en aquel entonces y esto poco a poco me estaba llevando, literalmente, a buscar alternativas.

Probe sistema de windows optimizados y ligeros e incluso instale versiones de windows “xp” pero siempre el resultado era el mismo.

Es asi que buscando una alternativa recordé, que, una vez conversando sobre sistemas operativos me nombraron que windows no era el único sistema operativo existente, sino que existía algo llamado “Ubuntu” que era un sistema basado en Linux. Al recordar eso fue que tomé la decisión de recopilar mas información para conocer si era una alternativa viable.

Realizando esta recopilación fue cuando me surgieron diferentes preguntas como:

  • ¿Podré realizar las mismas tareas que realizo en windows diariamente?
  • ¿Existirá alguna ventaja respecto a windows?
  • ¿Cómo se realizará el proceso de instalación del sistema?
  • ¿Tendré que adquirir algún software o licencia para su uso?
  • Y la mas importante de todas: ¿Valdrá la pena?
